# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-07
### Added
- Initial bare-metal implementation of the multi-sensor system.
- HC-SR04 ultrasonic distance tracking with microsecond DWT resolution.
- DHT11 custom 1-wire protocol implementation.
- LDR darkness detection logic.
- Hierarchical audiovisual alerts for reverse monitoring.
- UART telemetry interface (115200 baud).
- Comprehensive GitHub Actions CI pipeline and formatting configuration.
